# Workflow: Lanjutkan Portfolio WELLI

Gunakan workflow ini untuk melanjutkan pengembangan website portfolio Anda.

## Langkah-langkah:

### 1. Buka Project Folder
Pastikan Anda berada di folder project:
`C:\Users\HELLO\.gemini\antigravity\scratch\modern-portfolio`

### 2. Jalankan Development Server
// turbo
```bash
cd C:\Users\HELLO\.gemini\antigravity\scratch\modern-portfolio && npm run dev
```

### 3. Buka di Browser
Akses website di: http://localhost:5173/

## File-file Penting:
- **HTML**: `index.html` - Struktur halaman
- **CSS**: `src/style.css` - Semua styling
- **JS**: `src/main.js` - Interaktivitas (typewriter, scroll, dll)
- **Gambar Anda**: `public/images/about-me.png`
